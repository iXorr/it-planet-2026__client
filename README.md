## DESCRIPTION

Client-side for the Trampoline application, built for students and applicants to explore job opportunities and career events in a unified interface.

It provides list and map-based visualization of opportunities, with real-time interaction and role-based features for admins, employees, and applicants.

The map functionality is powered by the **Yandex Maps Open API**, enabling geographic visualization of events and job listings.

The entire application was developed as part of the **"IF... ELSE" 2026 hackathon** from **IT-PLANET** company.
